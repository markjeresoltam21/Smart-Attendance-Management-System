# Assets Folder

This folder contains app assets like icons and splash screens.

For Expo projects, you can use default assets or create your own:

## Required Assets

- **icon.png** - App icon (1024x1024px)
- **splash.png** - Splash screen (1242x2436px)
- **adaptive-icon.png** - Android adaptive icon (1024x1024px)
- **favicon.png** - Web favicon (48x48px)

## Using Default Assets

Expo will use default assets if these files don't exist. You can add custom assets later.

## Create Custom Assets

Use tools like:
- Figma
- Canva
- Adobe Illustrator
- Or use Expo's asset generator

For now, the app will work with Expo's default assets.
